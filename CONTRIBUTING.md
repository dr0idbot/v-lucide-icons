# Contributing

## Development Setup

- Java 21+
- Maven 3.9+

## Building

```shell
mvn clean install
```

## Running Tests

```shell
mvn test
```

## Updating Icons

See the [tools repository](https://github.com/dr0idbot/vaadin-lucide-icons) for the icon generator instructions.

## Code Style

- No Javadoc comments in source code
- Follow existing patterns in the codebase
- Keep the public API minimal

## Pull Requests

1. Ensure all tests pass
2. Update the CHANGELOG if adding features
3. Keep changes focused on a single concern
